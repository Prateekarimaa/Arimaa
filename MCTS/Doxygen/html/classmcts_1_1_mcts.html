Detailed Description

Mcts class.

Node* _root = adress of the node considered as the root of the tree. int _depth = depth to explore before running random simulations. int _simulationPerRoot = number of simulations to run per Root. int _simulationPerLeaves = number of simulations to run per Leaves. int _IAPlayer = player played by the algorithm (1 or 2).

mcts::Mcts::explore ( Node *  node,
int  depth 
)

if depth is > 0, fetch the possible moves and chose the best using UCT, and decrease depth by 1. if depth = 0, run playRandom if the node is a terminal, then update the stats :

  • set uct to 10 if it's a win
  • set parent's uct to -1 if it's a loss (we must never go to that node)
