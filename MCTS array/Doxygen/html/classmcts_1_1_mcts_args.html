Detailed Description

MctsArgs class.

options for the MCTS Algorithm int _depth = depth to explore before running random simulations. int _timeLimitsimulationPerRoot = time limit in ms int _simulationPerRoot = number of simulations to run per Root. int _simulationPerLeaves = number of simulations to run per Leaves. int _numberOfVisitBeforeExploration = number of visits on a node before expanding its children.
