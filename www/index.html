Fast & Furious Game Playing : Monte Carlo Drift

A MCTS Algorithm Artificial Intelligence software for Arimaa.

 
 

Monte Carlo Tree Search Algorithm


The purpose of this project is to create an Artificial Intelligence software for the game Arimaa able to compete against humans using the Monte Carlo Tree Search algorithm.

Due to the high number of branches (500 times more than chess), it is impossible to explore all the position.

Using probability, random simulations and backtracking, the
Monte Carlo Tree Search Algorithm provides a way to decrease the number of boards to be visited.
 

Tree Parallelisation...

on the multicore CPU.


This software has been optimized to take advantage of the power provided by the growing popularity of multicore computers.

Each core is working on its own part of the tree, multiplying the speed of the search up to 8 times !
 

Root Distribution...

on a set of clusters.

Should you be seeking for more power, you can unleash it on a set of cluster, taking advantage of the distribution of the simulations all over the network. Using a Master-Workers implementation each computer added to your cluster increase the odds of finding the best move.

As each worker has its own copy of the tree, it minimize the interaction between them, thus decreasing the inteferences and speeding up researches.
